# zkz project

这是一个基于竺可桢日记数据挖掘的Python入门项目.


## 数据来源

数据来源于[竺可桢日记](http://www.ncku1897.net/diary/)。

## 模块

- notes 里面的k1,k2等表示第几次分享内容
- datas 存放数据目录
- utils 一些通用函数
- dbs 存放数据库相关文件
